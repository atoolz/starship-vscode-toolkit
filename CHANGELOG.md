# Changelog

All notable changes to the Starship Toolkit extension will be documented in this file.

## [0.1.0] - 2026-03-20

### Added

- IntelliSense completions for all ~70 Starship modules
- Per-module option completions with type information and defaults
- Style string completions (bold, italic, colors, hex values)
- Format string variable completions per module context
- Hover documentation with descriptions, types, defaults, and doc links
- Diagnostics for unknown modules, unknown options, invalid styles, and type mismatches
- Snippets for starter config, common modules, and custom module templates
- Automatic activation for `starship.toml` files
